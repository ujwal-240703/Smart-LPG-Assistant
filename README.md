# 🔥 Smart LPG Assistant

An IoT-based LPG monitoring and safety system developed using the ESP32 microcontroller. The project continuously monitors LPG cylinder weight, detects gas leakage, and sends real-time alerts to improve household safety.

---

## 📖 Project Overview

LPG is one of the most commonly used fuels in households and industries. However, gas leakage and unexpected depletion of LPG cylinders pose significant safety and convenience challenges.

The Smart LPG Assistant addresses these problems by integrating an MQ-2 gas sensor, Load Cell with HX711 amplifier, and ESP32 microcontroller. The system detects gas leakage, measures the remaining LPG level, and provides real-time notifications using cloud connectivity.

## ⚙️ Working Principle

1. MQ-2 sensor continuously detects LPG concentration.
2. Load Cell measures the cylinder weight.
3. ESP32 processes sensor values.
4. Data is uploaded to Adafruit IO.
5. User receives gas level updates.
6. Gas leakage triggers immediate alerts.
